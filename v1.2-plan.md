# Sage v1.2 — The Holy Plan

This is the locked, prioritised plan for v1.2. Every PR should reference its
phase and update the status table at the bottom. Keep this file in sync with
reality — a plan you don't update is a wishlist.

## North star

> A local-first AI assistant for iPhone (15 Pro+) that is **demonstrably**
> trustworthy, action-capable, and visually intelligent — and stays useful
> all day under a 1–2 W sustained budget.

## The 13 non-negotiables (locked)

1. Hybrid specialists + controller (router, planner, writer, vision, ASR)
2. Deterministic action layer with hard safety rails (typed APIs, dry-run, rollback, audit)
3. Verifiable memory (3-tier, attribution, decay, user controls)
4. On-device retrieval that doesn't leak (per-source/per-query scope, encrypted)
5. Truth mode (calibrated confidence, citations, refusal without shame)
6. Inspectable privacy (air-gap toggle, audit log, encrypted at rest)
7. Screenshot + app state understanding
8. Shortcuts as first-class actuator
9. Style/voice cloning, bounded
10. Plan → Execute → Verify loop (closed-loop, not one-shot)
11. Battery/thermal budgeting + quality degradation
12. Offline updates without breaking trust (signed bundles, rollback)
13. Visual intelligence on-device (screenshot, doc-scan, semantic search, redaction)

## Ship-order

1. **Phase 0 → Phase 1** (foundations + actions): kills the regex parser, makes every action typed, previewable, undo-able.
2. **Phase 5** (screen understanding): the single most demoable feature.
3. **Phase 2 + Phase 4** (tiered memory + citations + truth mode): trustworthiness.
4. **Phase 3** (PEV controller): closes the loop.
5. **Phase 6–9**: polish month.

## Hard things to remember

- **Apple FoundationModels** requires iOS 26 + Apple Intelligence. Always carry a Llama 3.2 1B fallback router.
- **Florence-2 on MLX** isn't a maintained port. Phase 5 ships with SmolVLM-500M; port Florence-2 in a follow-up.
- **Screenshot capture** has no public iOS API. The flow is *user-initiated*: Share Extension or Action-Button shortcut → upload → analyze.
- **"Secure Enclave-encrypted"** is marketing-dishonest for SwiftData. Use `NSFileProtectionComplete` and describe it as "encrypted at rest, key bound to your passcode."
- **Style cloning ethically**: per-recipient opt-in, hard deny-list (children, employer's counsel, banking).
